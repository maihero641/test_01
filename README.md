# 🐍 스네이크 게임 (HTML 단일 파일)

키보드 **화살표 키(↑↓←→)** 로 플레이하는 클래식 스네이크 게임입니다.  
**HTML 한 파일** 안에 **CSS**와 **JavaScript**가 모두 포함되어 있어, 다운로드 후 바로 실행할 수 있습니다.

> 👉 게임 파일: `snake.html` (단일 파일)

---

## ✨ 기능 요약

- 단일 HTML 파일 구성
- 별도 설치 없이 브라우저에서 바로 실행
- 화살표 키 조작 지원
- `Space` 일시정지 / `Enter` 재시작
- 반응형 캔버스 기반 UI
- 사과 섭취 시 길이 증가 및 속도 상승
- `localStorage`를 이용한 최고 점수 저장

---

## 🎮 조작 방법

| 키 | 기능 |
|----|------|
| ↑ ↓ ← → | 이동 |
| Space | 일시정지 / 재개 |
| Enter | 게임 재시작 |

---
